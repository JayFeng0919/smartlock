# Smart Lock — 智能门锁

> STM32F103C8T6 + RC522 + AS608 指纹 + 4x4 键盘 + OLED

---

## 一、引脚接线

| 外设         | 接口       | STM32 引脚              |
|-------------|-----------|------------------------|
| OLED 0.96寸 | I2C       | PB8=SCL, PB9=SDA       |
| RC522 IC卡  | 软件 SPI   | PB1=RST, PA6=MISO, PA7=MOSI, PB10=SCK, PB11=NSS |
| 4x4 键盘    | 行列扫描    | 行 PA8~11, 列 PB12~15    |
| LED         | 推挽输出    | PC13 (低电平亮)           |
| 蜂鸣器       | 推挽输出    | PB5                     |
| AS608 指纹   | USART2    | PA4=电源, PA0=TouchOut, PA2=TX, PA3=RX |

键盘布局 (行列交叉):
```
      C1    C2    C3    C4
R1    1     2     3     A
R2    4     5     6     B
R3    7     8     9     C
R4    *     0     #     D
```

## 二、上电

OLED 显示:
```
Smart Lock
Ready
```

处于 **常态模式**，等待开锁。

## 三、开锁方式

### 刷卡
1. IC 卡靠近 RC522
2. 白名单内 → `True` + LED 亮 + 蜂鸣 1 声
3. 白名单外 → `False` + LED 闪 3 次 + 蜂鸣 3 声

### 密码
1. 常态下按任意数字键 → 该数字作为密码第 1 位
2. 继续输入后 3 位（共 4 位，出厂 `0000`）
3. 正确 → `True` + LED 亮 + 蜂鸣 1 声
4. 错误 → `False` + LED 闪 3 次 + 蜂鸣 3 声
5. 按 `*` 取消

### 指纹
1. 手指放到指纹模块上
2. 识别成功 → `Unlock OK` + LED 亮 + 蜂鸣 1 声
3. 识别失败 → `Unlock Failed` + LED 闪 3 次 + 蜂鸣 3 声
4. 无指纹或超时 → 自动返回常态

> 出厂指纹库为空，请先进入管理员模式添加。

---

## 四、管理员模式

1. 常态下长按 `#` 约 1.5 秒
2. 输入 4 位管理员密码（出厂 `0000`）
3. 正确 → 进入菜单
4. 连续 3 次错误 → 锁定 10 秒

管理员菜单:
```
1.Add  2.Del
3.View 4.PW 5.FP
```

| 按键 | 功能 |
|------|------|
| `1`  | 录入 IC 卡 |
| `2`  | 删除 IC 卡 |
| `3`  | 查看全部卡片（自动轮播） |
| `4`  | 修改管理员密码 |
| `5`  | 指纹管理 |
| `*`  | 退出 |

---

## 五、指纹管理

按 `5` 进入:
```
1.Add 2.ClearAll
3.Count  *.Back
```

| 按键 | 功能 |
|------|------|
| `1` | 添加一枚指纹（自动分配ID，无需手动输入） |
| `2` | 清空指纹库（二次确认：按 `1` 确认，`*` 取消） |
| `3` | 查看已录入数量 |
| `*` | 返回 |

- 添加指纹：显示 `Add Finger` → 手指放上模块 → 成功后显示 `Add Success!` + 当前总数
- 清空库：显示 `Clear All?` → 按 `1` 确认 → 发送空指令清空 Flash

## 六、蜂鸣器 / LED 反馈

| 场景               | 蜂鸣          | LED          |
|-------------------|-------------|-------------|
| 开锁成功            | 1 声 (200ms) | 常亮         |
| 开锁失败            | 3 声 (100ms) | 闪 3 次       |
| 录卡/添加指纹/清库成功 | 1 声 (200ms) | 常亮         |
| 录卡/添加指纹/清库失败 | —            | 闪 2 次       |
| 改密成功            | 2 声 (100ms) | —           |
| 管理员锁定           | 1 声 (500ms) | —           |

## 八、常见问题

**Q: 刷卡没反应？**
A: 检查 RC522 接线，确保 SPI 引脚正确。

**Q: OLED 不显示？**
A: 检查 I2C 接线 (PB8=SCL, PB9=SDA)。

**Q: 按键没反应？**
A: 检查矩阵键盘接线 (行 PA8~11, 列 PB12~15)。

**Q: 指纹模块 timeout？**
A: 检查 PA0/PA2/PA3/PA4 接线，确认 TX→RX, RX→TX 交叉连接，波特率 57600。用参考工程单独测试模块是否正常。

**Q: 只能识别第一个指纹？**
A: AS608 当前 N=1 参数限制，只比对第一个模板。如需多指纹，需修改 AutoIdentify 指令的搜索范围参数。

**Q: 密码忘了？**
A: 密码存 RAM，重新上电恢复为 `0000`。
