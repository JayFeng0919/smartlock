# Smart Lock — 智能门锁 用户操作手册

## 一、硬件说明
| 外设 | 接口 | 引脚 |
|------|------|------|
| OLED 显示屏 | 0.96寸 I2C | PB8=SCL, PB9=SDA |
| RC522 IC卡模块 | 软件 SPI | PB1=RST, PA6=MISO, PA7=MOSI, PB10=SCK, PB11=NSS |
| 4x4 矩阵键盘 | 行 PA8~11, 列 PB12~15 | 见下方布局 |
| LED 指示灯 | PC13 | 推挽输出 |
| 蜂鸣器 | PB5 | 推挽输出 |
| 指纹模块 | USART2 | PA4=电源, PA0=手指检测, PA2=TX, PA3=RX |

键盘布局：
```
    C1(PB15) C2(PB14) C3(PB13) C4(PB12)
R1(PA8)    1       2       3       A
R2(PA9)    4       5       6       B
R3(PA10)   7       8       9       C
R4(PA11)   *       0       #       D
```

## 二、上电启动
接通电源后，OLED 显示：
```
Smart Lock
Ready
```

此时处于**常态模式**，支持刷卡、密码、指纹三种方式开锁。

## 三、开锁方式

### 3.1 刷卡开锁
1. 将 IC 卡靠近 RC522 感应区
2. 白名单内的卡 → OLED 显示 `True`，LED 常亮，蜂鸣1声
3. 白名单外的卡 → OLED 显示 `False`，LED 快闪3下，蜂鸣3声

### 3.2 密码开锁
1. 直接按任意数字键 `0-9`
2. 该数字自动作为第 1 位密码，OLED 显示 `Password` + `*`
3. 继续输入后 3 位密码（共 4 位，默认 `0000`）
4. 密码正确 → OLED 显示 `True`，LED 亮，蜂鸣1声
5. 密码错误 → OLED 显示 `False`，LED 快闪3下，蜂鸣3声
6. 按 `*` 可取消密码输入

### 3.3 指纹开锁（新增）
1. 手指放到指纹模块上
2. 识别成功 → OLED 显示 `True`，LED 亮，蜂鸣1声
3. 识别失败 → OLED 显示 `False`，LED 快闪3下，蜂鸣3声
4. 超时未识别 → 自动返回常态界面
> 出厂时指纹库为空，请先进入管理员模式录入指纹。

## 四、进入管理员模式

1. 常态下长按 `#` 键约 1.5 秒
2. OLED 显示 `Password:`
3. 输入 4 位管理员密码（出厂默认 `0000`）
4. 密码正确 → 进入管理员菜单
5. 密码错误 3 次 → 锁定 10 秒，蜂鸣长鸣1声警告

## 五、管理员菜单

菜单显示：
```
1.Add  2.Del
3.View 4.PW 5.FP
```

| 按键 | 功能 |
|------|------|
| `1` | 录入 IC 卡 |
| `2` | 删除 IC 卡 |
| `3` | 查看已录入卡片 |
| `4` | 修改管理员密码 |
| `5` | 指纹管理 |
| `*` | 退出管理员模式 |

### 5.1 录入 IC 卡
1. 按 `1` 键 → OLED 显示 `Add Card` / `Place card...`
2. 将要录入的 IC 卡靠近 RC522
3. 录入成功 → OLED 显示 `Added!` + 卡号，蜂鸣1声
4. 卡已存在 → OLED 显示 `Already Exist`
5. 白名单满 → 显示 `List Full!`（最多 5 张）

### 5.2 删除 IC 卡
1. 按 `2` 键 → OLED 显示 `Del Card` / `Place card...`
2. 将要删除的 IC 卡靠近 RC522
3. 显示 `Sure?` → 按 `#` 确认 / 按 `*` 取消
4. 删除成功 → 显示 `Deleted!`，蜂鸣1声

### 5.3 查看已录入卡片
1. 按 `3` 键 → 自动轮播显示卡号
2. 按 `*` 返回

### 5.4 修改管理员密码
1. 按 `4` 键 → 输入旧密码 → 输入新密码 → 再次确认
2. 两次一致 → 显示 `Changed!`，蜂鸣2声
3. 不一致 → 显示 `Mismatch!`

### 5.5 指纹管理（新增）
按 `5` 进入指纹管理子菜单：
```
1.Enroll 2.Del
3.Count  *.Back
```

| 按键 | 功能 |
|------|------|
| `1` | 录入指纹 |
| `2` | 删除指纹 |
| `3` | 查看已录入数量 |
| `*` | 返回上级菜单 |

#### 录入指纹
1. 按 `1` → 显示 `Enroll Finger`
2. 输入 2 位 ID 号（01~50）
3. 手指放到模块上按提示操作
4. 成功显示 `Enroll OK!`，蜂鸣1声

#### 删除指纹
1. 按 `2` → 显示 `Delete Finger`
2. 输入 2 位 ID 号
3. 成功显示 `Deleted!`，蜂鸣1声

#### 查看数量
1. 按 `3` → 显示 `Finger Count:` + 当前已录入数

## 六、蜂鸣器反馈

| 场景 | 蜂鸣行为 |
|------|---------|
| 开锁成功（刷卡/密码/指纹） | 1 声长鸣 (200ms) |
| 开锁失败（刷卡/密码/指纹） | 3 声短鸣 (各 100ms) |
| 管理员密码锁定 | 1 声长鸣 (500ms) |
| 录卡/删卡/录指纹/删指纹成功 | 1 声 (200ms) |
| 修改密码成功 | 2 声 (各 100ms) |

## 八、常见问题
**Q: 刷卡没反应？**
A: 检查 RC522 接线，确保 SPI 引脚正确。

**Q: OLED 不显示？**
A: 检查 I2C 接线（PB8=SCL, PB9=SDA）。

**Q: 按键没反应？**
A: 检查矩阵键盘接线（行 PA8~11, 列 PB12~15）。

**Q: 指纹模块没反应？**
A: 检查 PA0/PA2/PA3/PA4 接线，确认模块供电正常（PA4 高电平 3.3V），串口波特率 57600。

**Q: 密码忘了怎么办？**
A: 密码存 RAM，重新上电恢复为 `0000`。

**Q: 蜂鸣器不响？**
A: 检查 PB5 是否接好，有源蜂鸣器正极接 PB5，负极接 GND。
