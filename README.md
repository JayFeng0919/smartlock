# Smart Lock — 智能门锁 用户操作手册

## 一、硬件说明
| 外设 | 接口 | 引脚 |
|------|------|------|
| OLED 显示屏 | 0.96寸 I2C | PB8=SCL, PB9=SDA |
| RC522 IC卡模块 | 软件 SPI | PB1=RST, PA6=MISO, PA7=MOSI, PB10=SCK, PB11=NSS |
| 4x4 矩阵键盘 | 行 PA8~11, 列 PB15~12 | 见下方布局 |
| LED 指示灯 | PA0 | 推挽输出 |
| 蜂鸣器 | PB5 | 推挽输出 |
| 指纹模块(预留) | PA2=TX, PA3=RX | 暂未接 |

键盘布局：
```
    C1(PB15) C2(PB14) C3(PB13) C4(PB12)
R1(PA8)    1       2       3       A
R2(PA9)    4       5       6       B
R3(PA10)   7       8       9       C
R4(PA11)   *       0       #       D
```

## 二、上电启动
接通电源后，OLED 显示：
```
Smart Lock
Ready
```

此时处于**常态模式**，支持刷卡或密码开锁。

## 三、刷卡开锁
1. 将 IC 卡靠近 RC522 感应区
2. 白名单内的卡 → OLED 显示 `True`，LED 常亮，蜂鸣1声
3. 白名单外的卡 → OLED 显示 `False`，LED 快闪3下，蜂鸣3声
> **注意**：出厂时白名单为空，任何卡都会显示 False。请先进入管理员模式录入卡片。

## 四、密码开锁
1. 在常态模式下，直接按任意数字键 `0-9`
2. 该数字自动作为第 1 位密码，OLED 显示 `Password` + `*`
3. 继续输入后 3 位密码（共 4 位，默认 `0000`）
4. 密码正确 → OLED 显示 `True`，LED 亮，蜂鸣1声
5. 密码错误 → OLED 显示 `False`，LED 快闪3下，蜂鸣3声
6. 按 `*` 可取消密码输入
> 密码和管理员密码共用一个，修改密码需进入管理员模式。

## 五、进入管理员模式

1. 在常态模式下，长按 `#` 键约 1.5 秒
2. OLED 显示 `Password:`
3. 输入 4 位管理员密码（出厂默认 `0000`）
4. 密码正确 → 进入管理员菜单
5. 密码错误 3 次 → 锁定 10 秒，蜂鸣长鸣1声警告

## 六、管理员菜单

菜单显示：
```
1.Add  2.Del
3.View 4.PW
```

按键操作：
| 按键 | 功能 |
|------|------|
| `1` | 录入 IC 卡 |
| `2` | 删除 IC 卡 |
| `3` | 查看已录入卡片 |
| `4` | 修改管理员密码 |
| `*` | 退出管理员模式 |

### 6.1 录入 IC 卡
1. 按 `1` 键选择录入
2. OLED 显示 `Add Card` / `Place card...`
3. 将要录入的 IC 卡靠近 RC522
4. 录入成功 → OLED 显示 `Added!` + 卡号，蜂鸣1声
5. 卡已存在 → OLED 显示 `Already Exist`
6. 白名单满 → OLED 显示 `List Full!`（最多 5 张）

### 6.2 删除 IC 卡
1. 按 `2` 键选择删除
2. OLED 显示 `Del Card` / `Place card...`
3. 将要删除的 IC 卡靠近 RC522
4. OLED 显示 `Sure?` / `#=Yes *=No`
5. 按 `#` 确认删除 → OLED 显示 `Deleted!`，蜂鸣1声
6. 按 `*` 取消 → 返回菜单

如果刷的卡不在白名单中，显示 `Not in list`。

### 6.3 查看已录入卡片
1. 按 `3` 键选择查看
2. OLED 显示当前卡片序号和 4 字节卡号
3. 自动轮播切换卡片（每张显示约 1.5 秒）
4. 按 `*` 返回菜单

### 6.4 修改管理员密码
1. 按 `4` 键选择修改密码
2. 输入旧密码（4 位数字，用 `*` 遮盖）
3. 输入新密码（4 位数字）
4. 再次输入新密码确认
5. 两次一致 → OLED 显示 `Changed!`，蜂鸣2声
6. 两次不一致 → OLED 显示 `Mismatch!`

## 七、蜂鸣器反馈

| 场景 | 蜂鸣行为 |
|------|---------|
| 开锁成功（刷卡/密码） | 1 声长鸣 (200ms) |
| 开锁失败（刷卡/密码） | 3 声短鸣 (各 100ms) |
| 管理员密码锁定 | 1 声长鸣 (500ms) |
| 录卡/删卡成功 | 1 声 (200ms) |
| 修改密码成功 | 2 声 (各 100ms) |

## 八、功能摘要
| 场景 | 操作 |
|------|------|
| 刷卡开锁 | 常态下刷白名单内 IC 卡 |
| 密码开锁 | 常态下按任意数字键 → 输 4 位密码 |
| 进入管理 | 常态下长按 `#` 1.5 秒 → 输管理员密码 |
| 录入新卡 | 管理 → `1` → 刷卡 |
| 删除卡片 | 管理 → `2` → 刷卡 → `#` 确认 |
| 查看卡片 | 管理 → `3` → 自动轮播 |
| 修改密码 | 管理 → `4` → 旧密码 → 新密码 → 确认 |
| 退出管理 | 菜单中按 `*` |
| 取消操作 | 密码输入中按 `*` |

## 九、常见问题
**Q: 刷卡没反应？**
A: 检查 RC522 接线，确保 SPI 引脚正确。

**Q: OLED 不显示？**
A: 检查 I2C 接线（PB8=SCL, PB9=SDA）。

**Q: 按键没反应？**
A: 检查矩阵键盘接线（行 PA8~11, 列 PB15~12）。

**Q: 密码忘了怎么办？**
A: 目前密码存 RAM，重新上电恢复为 `0000`。后续加入 Flash 存储后，需通过代码重置。

**Q: LED 不亮？**
A: 检查 PA0 是否接好，LED 串联 220Ω 限流电阻到 GND。

**Q: 蜂鸣器不响？**
A: 检查 PB5 是否接好，有源蜂鸣器正极接 PB5，负极接 GND。
